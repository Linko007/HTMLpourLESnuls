Mais également à associer le CSS à vos pages HTML mais avant tout cela, voyons ensemble deux/trois définitions.

L'HTML c'est quoi ?

HTML est un langage de programmation informatique utilisé pour créer la structure et le contenu
de pages web. Il s'agit d'un langage de balisage, ce qui signifie qu'il utilise des balises
(tags) pour définir différents éléments sur une page web, tels que des titres,
des paragraphes, des liens, des images, des formulaires, etc. Ces balises permettent de structurer
et de formater le contenu de la page de manière cohérente.

ET le CSS alors ?

CSS (Cascading Style Sheets) est un langage de programmation utilisé pour définir la présentation et le style d'un document HTML
Il permet de contrôler la mise en forme, l'apparence et la disposition des éléments sur une page web.
En d'autres termes, CSS permet de définir la façon dont les éléments HTML sont affichés à l'écran.

Maintenant que nous savons ce qu'est l'HTML et le CSS, nous pouvons passer à la suite.
