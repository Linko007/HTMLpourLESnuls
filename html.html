< head> :

La balise < head> est une balise HTML qui est utilisée pour contenir des
informations sur le document HTML, mais qui ne sont pas directement visible sur la page web.

On y retrouve les balises < meta> que l'on va passé, et également la balise < title> .
La balise < title> est le titre de votre page, il sera afficher dans les l'onglets sur votre navigateur web.

< body> :

La balise < body> en HTML est utilisée pour définir le contenu principal d'une page web.
C'est à l'intérieur de cette balise que vous placez tout ce que les visiteurs verront sur votre site.
En d'autres termes, le contenu que vous mettez entre les balises < body> sera affiché à l'écran.

C'est donc ici que nous créerons notre pages à partire de balise (texte, titres, section,...)

1.2.Les Balises de Texte :

< h1>, < h2>, < h3>, ...

Les balises < h1>, < h2>, < h3>, et ainsi de suite, sont utilisées pour définir les titres et les
sous-titres d'une page HTML. Elles sont essentielles pour organiser et hiérarchiser le contenu de manière
claire et compréhensible.

< p>

La balise < p> est utilisée pour définir des paragraphes de texte dans une page HTML

< a>

La balise < a> en HTML est utilisée pour créer des liens hypertextes conduisant donc vers d'autres
pages web, des fichiers, des adresses email, ou même d'autres emplacements sur la même page.

< strong>

La balise < strong> est utilisée pour indiquer du texte qui doit être mis en évidence, c'est-à-dire du texte important
ou significatif qui doit être rendu en gras.

1.3.Les Balises de sections :

< header>

La balise < header> est utilisée pour définir l'en-tête d'une section ou de la page entière. Elle contient généralement des éléments tels que des titres, des
logos, des menus de navigation, etc.

< footer>

La balise < footer> est utilisée pour définir le pied de page d'une section ou de la page entière. Elle contient
souvent des informations de copyright, des liens vers des pages connexes, ou des informations de contact.

< nav>

La balise < nav> est utilisée pour définir une section de navigation. Elle contient généralement des liens vers d'autres pages de votre site web.

< section>

La balise < section> est utilisée pour définir des sections génériques ou des divisions thématiques dans le contenu. Elle est utile
lorsque le contenu de la page est divisé en parties distinctes qui ont leur propre en-tête, paragraphe, etc.

< div>

La balise < div> est une balise générique utilisée pour diviser le contenu en conteneurs. Elle est souvent utilisée avec des classes ou des identifiants CSS pour appliquer
des styles ou des comportements spécifiques.

2.2.Les Tableaux :

Les tableaux sont utilisés pour organiser les données sous forme de lignes et de colonnes, de la même manière qu'un tableau que l'on
pourrait dessiner sur du papier.

Structure de base d'un tableau :

< table >

La balise < table > sert à définir le tableau.

< tr >

La balise < tr > sert à définir une ligne du tableau.

< th >

La balise < th > sert à définir une cellule d'en-tête, elle est utilisé pour indiquer le contenu de chaque colonne.
Les éléments < th > sont généralement affichés en gras et centré par défaut.

< td >

La balise < td > sert à définir une cellule de données, les données de tableau contiennent les informations rélles que vous
souhaitez afficher dans le tableau. Les élément < td > sont généralement affichés en texte normal

3.Image et Média

3.1.Les Images :

Une image est un élément visuel qui peut être affiché sur une page web pour montrer des photos, des graphiques ou des illustrations.

< img >

La balise < img > est utilisé pour afficher des images sur une page web.

Voici la structure du code sur VS Code :

"src" : désigne l'emplacement de votre image (ici c'est "img/thomas.jpeg"). L'image doit être stocké dans le même
dossier que la page HTML (souvent dans un dossier image).

"alt" : texte alternatif qui s'affiche si l'image ne peut pas être chargée (une déscription de l'image).

"title" : title est un ajout utile lorsque l'on passe notre souris sur l'image, pour affiché un texte. Tout comme l'atribut alt
il décrit l'image (mais en moins gros)

en ligne ou à une adresse email.
